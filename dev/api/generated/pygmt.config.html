Change GMT default settings globally or locally.

Change GMT default settings globally:

pygmt.config(PARAMETER=value)

Change GMT default settings locally by using it as a context manager:

with pygmt.config(PARAMETER=value):
    ...

Full GMT defaults list at https://docs.generic-mapping-tools.org/6.6/gmt.conf.html.

Note

PS_CONVERT is not supported. To configure conversion options, please pass parameters to pygmt.Figure.savefig or pygmt.Figure.show instead.
